Project Name
Pengembangan Aplikasi UMKM Kota Semarang
Project Description
Pengembangan Aplikasi UMKM Kota Semarang dibuat untuk memberikan kemudahan bagi masyarakat yang bergerak di bidang bisnis terutama pada UMKM. Hal ini didasarkan pada masih banyak masyarakat yang belum memanfaatkan teknologi pemasaran seperti E-commerce dan lainnya. Penelitian ini menggunakan model pengembangan SDLC. Model analisisnya menggunakan Use Case, Activity, Class Diagram, E-R Diagram dan kamus data. Manfaatnya dapat memberikan kemudahan dalam memasarkan produk UMKM kota semarang agar lebih dikenal oleh masyarakat lokal maupun luar daerah.
